## 项目概述

Pulse 是一个基于 Spring Boot 的微服务架构健身应用，专注于为用户提供个性化的运动和健康管理服务。

### 核心功能
- 用户管理和认证
- 运动标签和个性化推荐
- AI智能健身建议
- 签到和积分系统
- 后台管理系统

### 技术栈
- Spring Boot 2.7.4
- MyBatis
- Redis
- MySQL 8.0
- JWT认证
- Docker容器化
